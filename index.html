Publication

Implementing and Automating Security Scanning to a DevSecOps CI/CD Pipeline
IEEE, World Conference on Communication & Computing (WCONF 2023)

The Research Paper discusses the impact of DevOps, containerization, and CI/CD on software development, highlighting security challenges in containerized applications. It emphasizes the significance of security scanning in DevSecOps pipelines and proposes using Snyk and StackHawk tools for automated vulnerability scanning and fixing. The research aims to assess the effectiveness of this approach, asserting its potential to enhance overall application security by reducing vulnerability detection and fixing times.

IEEE Xplore Link

Security Testing of Containerized IoT Applications in the Automatic Deployment Process
IEEE, International Interdisciplinary Humanitarian Conference for Sustainability (IIHC 2023)

The paper explores the security challenges in automatically deploying containerized IoT applications amidst the rapid growth of containerization and the Internet of Things (IoT). It proposes integrating Snyk, Trivy, and Anchore security scanning tools into GitHub Actions workflows for security testing. The methodology includes configuring automated deployment pipelines and assessing the tools' effectiveness in identifying vulnerabilities. The research presents a comparative analysis of the tools, emphasizing accuracy, speed, and coverage. It concludes by offering practical insights on seamless tool integration, providing valuable guidance for IoT developers and DevOps teams to enhance security in automatic deployments, fostering the development of secure IoT ecosystems.
